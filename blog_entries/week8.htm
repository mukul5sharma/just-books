Project Description 

My project has two parts-
Part 1 is about trading used books between users that is a user can ad a book that he wants to sell 
and another user can contact this user and buy the book.   

Part 2 includes some social networking for Book lovers where a user can like a book, rate a book, 
follow other users, create his profile, receive updates from other users and post reviews for Books.
            
I will be using Google books api for creating book profiles and for implementing book search.
